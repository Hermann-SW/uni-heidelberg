
## Hausaufgabe 5.2b

Anstelle eines normalen Beweises mit Fallunterscheidung,  
automatische Bestimmung aller Gruppen mit 4 Elementen  
mittels Python script [G4.py](scripts/G4.py):
```
$ python G4.py 
0 1 2 3
1 0 3 2
2 3 0 1
3 2 1 0

0 1 2 3
1 0 3 2
2 3 1 0
3 2 0 1

0 1 2 3
1 2 3 0
2 3 0 1
3 0 1 2

0 1 2 3
1 3 0 2
2 0 3 1
3 2 1 0

$
```
